About


The team:

Kathy Agafonov & Netanel Krotovich.

We chose to develop a game inspired by Chicken Invaders, where our main role is Rick, trying to kill "Scary Terry".

This game was developed using JavaScript, HTML5 and CSS. We used the Validate jQuery Plugin and slim.min.js package, a Date Picker Template, designed forms and buttons, and the W3.CSS framework and Googleapis fonts.

The difficulties in this work were:

  1. Make the characters move and connect them to listeners.
  2. Styling the different pages of the game.
  3. Developing the validation using jQuery.

Game Instructions

Game Keys:

  Up: use the ⬆ Key
  Down: use the ⬇ Key
  Left: use the ⬅ Key
  Right: use the ➡ Key
  Shot: Space Bar by deafult, but you can choose your own Key

General instructions:

  • To start a game you have to have a user first and to be logged in.
  • After login, you can choose the shooting key and game duration. Default shooting key is space and 2 minutes.
  • You are starting the game with 3 lives. After 3 hits it's game over and you lose the game.
  • During playing the game, in the right corner you can see the time left to play.
  • Notice! If time is over and you earned less than 100 points - you lose!
  • In the right top corner, you can see the number of lives you have left.
  • There's a button alowing to mute and unmute the music playing during the game.
  • There's a button to stop and resume the game while playing.

Defeat Scary Terry and rescue Morty from the dream invader!
